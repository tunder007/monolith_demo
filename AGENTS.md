# AGENTS.md

Cross-tool entry point for AI coding agents (Codex, Cursor, etc.). This repo's
guidance is maintained in one place to avoid drift:

- **[CLAUDE.md](./CLAUDE.md)** — the quick map: what the project is, commands,
  layout, and the rules to follow.
- **[docs/](./docs/README.md)** — the authoritative documentation (architecture,
  roadmap, CLI spec, package contracts).
- **[docs/DECISIONS.md](./docs/DECISIONS.md)** — the tiebreaker; authoritative when
  anything conflicts.

Read `CLAUDE.md` first, then the relevant `docs/` page for the area you're
changing. The same rules apply to every agent regardless of tool.
